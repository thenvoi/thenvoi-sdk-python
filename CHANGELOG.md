## [Unreleased]

### ⚠ BREAKING CHANGES

* **tools:** Platform tool names now have `thenvoi_` prefix to prevent naming collisions with custom tools:
  - `send_message` → `thenvoi_send_message`
  - `send_event` → `thenvoi_send_event`
  - `add_participant` → `thenvoi_add_participant`
  - `remove_participant` → `thenvoi_remove_participant`
  - `lookup_peers` → `thenvoi_lookup_peers`
  - `get_participants` → `thenvoi_get_participants`
  - `create_chatroom` → `thenvoi_create_chatroom`

  **Migration:** Update any code that references tool names directly. The `get_tool_description()` function still accepts unprefixed names but will emit a deprecation warning.

  **Note:** Claude SDK MCP names (`mcp__thenvoi__*`) are unchanged since the MCP server name already provides namespacing.

## [0.1.0] - 2024-01-01

### Added

- Initial SDK implementation for Thenvoi platform
- ThenvoiLink for WebSocket + REST transport
- AgentRuntime for convenient agent lifecycle management
- RoomPresence for cross-room lifecycle management
- ExecutionContext for per-room context accumulation
- AgentTools for platform tools (send_message, add_participant, etc.)
- Support for multiple AI frameworks (PydanticAI, Anthropic, LangGraph, Claude SDK)
- Comprehensive test suite with pytest
